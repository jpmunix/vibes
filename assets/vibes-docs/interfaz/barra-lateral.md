# Barra Lateral

La barra lateral izquierda es tu centro de navegación. Su contenido cambia según la sección activa.

## Lista de Apps

![Barra lateral con apps](https://images.mnstatic.com/Tools/files/af1143cdcdb15e885f5a0ccd196e83009d58fbf4698b08a60ec38dfd19d4de19.jpg)
<!-- @info title="📸 Captura pendiente" "Reemplazar con captura de la barra lateral mostrando la lista de apps" -->

Cuando estás en la sección **Apps**, la barra lateral muestra:

- **Tus aplicaciones** ordenadas por actividad reciente
- **Barra de búsqueda** para filtrar por nombre
- **Menú contextual** (click derecho) con opciones: renombrar, archivar, eliminar, compartir

### Menú de acciones

En la parte superior de la lista hay un menú desplegable con acciones rápidas:

| Acción | Qué hace |
|---|---|
| Nueva aplicación | Crea un proyecto vacío desde plantilla |
| Importar carpeta | Abre un proyecto existente de tu disco |
| Buscar | Activa el filtro de búsqueda |
| Cerrar todas | Cierra todas las apps abiertas |

## Lista de Chats (dentro de una app)

Al entrar en una app, la barra lateral se transforma en la **lista de chats**:

- **Chats activos** con su título, última actividad y etiquetas
- **Chats fijados** (📌) siempre visibles arriba
- **Chats archivados** accesibles desde un desplegable

### Selector de chat

El selector de chat se ha diseñado con acciones directas visibles:

- **Resumir**: condensa la conversación y abre un chat limpio con el resumen como contexto
- **Archivar**: mueve el chat fuera de la vista principal
- **Eliminar**: borra el chat (la memoria relevante se extrae automáticamente antes)

### Etiquetas de color

Cada chat puede tener una o varias **etiquetas con color**. Se añaden desde el menú contextual:

1. Click derecho sobre un chat
2. Selecciona **Añadir etiqueta**
3. Elige nombre y color de la paleta
4. La etiqueta aparece como un badge debajo del título

Para eliminar una etiqueta, pasa el cursor por encima y haz click en la ✕.

<!-- @tip "Las etiquetas funcionan en chats normales, fijados y archivados. Úsalas para categorizar por tema: 'bug', 'diseño', 'backend', etc." -->

## Redimensionar

La barra lateral es **redimensionable**: arrastra el borde derecho para ajustar el ancho. El ancho se guarda automáticamente entre sesiones.
