# Vista de Detalles de una App

Cuando seleccionas una aplicación, se abre la **vista de detalles** que combina el chat con las herramientas de desarrollo.

## Cabecera de la app

En la parte superior de la vista de detalles encontrarás:

- **Nombre de la app** (editable haciendo click)
- **Selector de modelo**: permite cambiar el modelo de IA para este chat
- **Selector de modo**: cambia entre Agent, Plan y Ask
- **Indicador de artefactos**: badge azul si hay documentos/planes generados
- **Coste de sesión**: si tienes activado el gasto visible

## Panel de chat

El panel central es donde ocurre la conversación con el agente. Ver la sección **Chat y Agente IA** para detalles completos.

## Panel de vista previa

A la derecha (o izquierda, configurable) aparece el **panel de vista previa** donde se muestra tu aplicación ejecutándose en tiempo real.

### Controles de la vista previa

| Control | Función |
|---|---|
| **URL** | Dirección local de la app (editable para navegar) |
| **Recargar** | Fuerza una recarga de la vista previa |
| **Abrir en navegador** | Abre la app en tu navegador por defecto |
| **Herramientas** | Acceso al visor de código, terminal, Git, consola, base de datos |

### Configurar la posición

Puedes elegir si la vista previa se abre a la **derecha** o a la **izquierda** del chat desde **Ajustes → Flujo de trabajo → Expandir vista previa**.

También puedes configurar si se abre automáticamente cuando hay cambios o solo manualmente.

## Barra de herramientas

Debajo de la vista previa hay una barra de herramientas con acceso directo a:

| Herramienta | Icono | Descripción |
|---|---|---|
| **Código** | 📄 | Explorador de archivos del proyecto |
| **Terminal** | ⌨️ | Consola de sistema del proyecto |
| **Git** | 🔀 | Panel de control de versiones |
| **Consola** | 📋 | Logs del servidor de desarrollo |
| **Base de datos** | 🗄️ | Visor de base de datos (si aplica) |
| **Memorias** | 🧠 | Panel de memoria del proyecto |

Cada herramienta se abre en su propia **ventana secundaria** dedicada.
