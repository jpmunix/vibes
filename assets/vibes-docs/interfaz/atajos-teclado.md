# Atajos de Teclado

Referencia rápida de los atajos más útiles en Vibes.

## Generales

| Atajo | Acción |
|---|---|
| <!-- @kbd "Ctrl+R" --> | Recargar la vista/ventana actual |
| <!-- @kbd "F5" --> | Recargar la vista/ventana actual |
| <!-- @kbd "Ctrl+Shift+I" --> | Abrir DevTools (debugging) |
| <!-- @kbd "Ctrl++" --> | Aumentar zoom |
| <!-- @kbd "Ctrl+-" --> | Reducir zoom |
| <!-- @kbd "Ctrl+0" --> | Restablecer zoom |

## Chat

| Atajo | Acción |
|---|---|
| <!-- @kbd "Enter" --> | Enviar mensaje |
| <!-- @kbd "Shift+Enter" --> | Nueva línea en el mensaje |
| <!-- @kbd "Escape" --> | Detener la generación del agente |

## Navegación

| Atajo | Acción |
|---|---|
| <!-- @kbd "Ctrl+," --> | Abrir Ajustes |

<!-- @tip "Los atajos funcionan tanto en la ventana principal como en las ventanas secundarias." -->
