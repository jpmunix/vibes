# Git Integrado

Vibes incluye un sistema completo de control de versiones Git sin necesidad de usar la terminal.

## Commits automáticos

Si tienes activado **Confirmar cambios en Git** en **Ajustes → Flujo de trabajo**, el agente hará commits automáticos después de cada cambio significativo. Cada commit incluye un mensaje descriptivo generado automáticamente.

## Panel de Git

Accede al panel de Git desde la barra de herramientas de la vista previa. Se abre en una ventana secundaria dedicada.

### Funcionalidades

| Función | Descripción |
|---|---|
| **Historial de commits** | Lista cronológica de todos los cambios |
| **Diff visual** | Compara archivos modificados lado a lado |
| **Branches** | Crear, cambiar y fusionar ramas |
| **Push/Pull** | Sincronizar con repositorio remoto (requiere GitHub) |

## Integración con GitHub

Conecta tu cuenta de GitHub desde **Ajustes → Integraciones → GitHub** para:

- Push automático a repositorios remotos
- Crear y gestionar pull requests
- Clonar repositorios existentes
