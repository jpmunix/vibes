# Base de Datos

Vibes incluye un visor de base de datos para proyectos que usen Supabase, PocketBase o conectores similares.

## Acceso

Click en el icono de base de datos (🗄️) en la barra de herramientas de la vista de detalles.

## Funcionalidades

- **Lista de tablas** en el panel izquierdo
- **Visor de datos** con paginación en el panel derecho
- **Filtros y búsqueda** dentro de las tablas

## Bases de datos soportadas

| Servicio | Cómo conectar |
|---|---|
| **Supabase** | Desde Ajustes → Integraciones → Supabase |
| **Neon** | Desde Ajustes → Integraciones → Neon |
| **PocketBase** | Detección automática si el proyecto lo incluye |
