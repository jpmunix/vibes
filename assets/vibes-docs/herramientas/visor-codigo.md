# Visor de Código

El visor de código te permite explorar y leer los archivos de tu proyecto sin salir de Vibes.

## Acceso

Click en el icono de código (📄) en la barra de herramientas de la vista de detalles.

## Funcionalidades

### Explorador de archivos

- **Árbol de directorios** colapsable en el panel izquierdo
- Arranca con todos los directorios **colapsados por defecto**
- Botones de utilidad:
  - **Colapsar todo**: vuelve al estado inicial
  - **Expandir primer nivel**: muestra solo los directorios raíz

### Iconos por tipo

Cada archivo muestra un icono coloreado según su tipo:

| Color | Tipos de archivo |
|---|---|
| 🔵 Azul | TypeScript (.ts, .tsx) |
| 🟡 Amarillo | JavaScript (.js, .jsx) |
| 🩷 Rosa | CSS, SCSS, Sass |
| 🟣 Indigo | PHP |
| 🟢 Verde | JSON, YAML |

### Visor de contenido

Al seleccionar un archivo, se muestra su contenido con **resaltado de sintaxis** completo en el panel derecho.
