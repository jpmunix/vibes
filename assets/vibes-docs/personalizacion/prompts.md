# Prompts Personalizados

Vibes permite personalizar las instrucciones internas que reciben los modelos de IA. Esto te da control total sobre el comportamiento del agente.

## Acceso

Desde **Ajustes → Prompts** puedes editar:

### Prompt del sistema

Las instrucciones base que recibe el agente al iniciar cualquier conversación. Aquí puedes incluir:

- Reglas de estilo de código
- Convenciones del proyecto
- Restricciones técnicas
- Idioma preferido para las respuestas

### Prompt de nombres

Las instrucciones que usa el modelo ejecutor para generar nombres de apps y chats.

### Prompt de memoria

Las instrucciones que guían al agente sobre qué información guardar en memoria y cómo organizarla.

## Idioma del chat

Desde **Ajustes → Agente → Idioma del chat** puedes seleccionar el idioma en el que responde el agente:

- Español
- English

El agente respetará este idioma en todas sus respuestas, independientemente del idioma del prompt del usuario.
