# Tipografía

Vibes permite personalizar las fuentes de la interfaz y del chat de forma independiente.

## Fuente de la interfaz

Controla la tipografía de **menús, botones, barras laterales y paneles**. Se configura desde **Ajustes → Tema → Tipografía de la Interfaz**.

Fuentes disponibles: Inter, Roboto, Outfit, Open Sans, Lato, Poppins, Nunito, Source Sans Pro y más.

## Fuente del chat

Controla la tipografía de **los mensajes del chat y las respuestas del agente**. Se configura desde **Ajustes → Tema → Tipografía del Chat**.

Puedes usar una fuente más legible para texto largo (como Georgia o Merriweather) y una más compacta para la interfaz.

## Escalas de tamaño

Desde **Ajustes → Tema → Tamaño de fuente** puedes ajustar tres escalas independientes:

| Escala | Qué afecta |
|---|---|
| **Interfaz** | Menús, botones, paneles |
| **Sidebar** | Listas de apps y chats |
| **Chat** | Mensajes y respuestas |

Cada escala va de 0.8× a 1.3× con incrementos de 0.05.

## Ancho de burbuja

También puedes controlar el **ancho máximo de las burbujas del chat** para ajustar la longitud de línea a tu preferencia de lectura.
