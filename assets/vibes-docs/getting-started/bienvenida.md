# Bienvenida a Vibes

Vibes es tu entorno de desarrollo potenciado por inteligencia artificial. Diseñado para que puedas crear, desarrollar y desplegar aplicaciones de forma integral sin salir de una sola herramienta.

## ¿Qué puedes hacer con Vibes?

- **Crear aplicaciones** desde plantillas o desde cero con asistencia de IA
- **Chatear con modelos de IA** que entienden tu código y tu proyecto
- **Previsualizar en tiempo real** los cambios mientras desarrollas
- **Gestionar versiones** con Git integrado
- **Desplegar** a Vercel, Supabase o Firebase con un solo click

## Filosofía

Vibes no es solo un editor de código. Es un **compañero de desarrollo** que aprende tus preferencias, sugiere mejoras y te ayuda a mantener la consistencia en tu proyecto a través de su Base de Conocimientos integrada.

> Empieza por la sección de **Configuración Inicial** para preparar tu entorno de trabajo.
