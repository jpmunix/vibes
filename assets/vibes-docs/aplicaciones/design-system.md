# Sistemas de Diseño

Vibes permite asignar un **sistema de diseño** a tus proyectos. Esto genera un archivo `DESIGN.md` que el agente consulta como referencia visual para todo el código que genere.

## ¿Qué es el DESIGN.md?

Es un archivo que contiene directrices de diseño: colores, tipografías, espaciado y componentes visuales. El agente lo usa como guía para generar código coherente con el estilo que elijas.

## Formas de asignar un diseño

### Diseños predefinidos

Vibes incluye una biblioteca de diseños de marcas populares. Selecciona uno antes de crear tu app y se descargará automáticamente.

### Pegar contenido

Puedes pegar directamente el contenido de un archivo de diseño personalizado.

### Generar por visión

Sube una **captura de pantalla** de un diseño que te guste y la IA analizará los colores, tipografías, espaciado y componentes para generar automáticamente un DESIGN.md basado en la imagen.

## Descargar el DESIGN.md

Desde la vista de detalles de cualquier app puedes **descargar el DESIGN.md activo** en cualquier momento para reutilizarlo en otros proyectos.
