# Crear una Aplicación

Vibes ofrece múltiples formas de iniciar un nuevo proyecto.

## Desde la pantalla Home

La forma más directa: escribe lo que quieres crear en el campo de texto de la pantalla principal y pulsa <!-- @kbd "Enter" -->.

1. Vibes genera un nombre para tu proyecto automáticamente
2. Crea la estructura del proyecto con la plantilla seleccionada
3. Abre una ventana de chat donde el agente empieza a trabajar
4. Arranca el servidor de desarrollo

## Desde el menú de Apps

1. Haz click en el menú **▾** de la pestaña Apps
2. Selecciona **Nueva Aplicación**
3. Se creará un proyecto vacío con la plantilla por defecto

## Importar un proyecto existente

Si ya tienes un proyecto en tu máquina:

1. Haz click en el menú **▾** de la pestaña Apps
2. Selecciona **Importar Carpeta**
3. Elige la carpeta de tu proyecto
4. Vibes detectará automáticamente el framework y configurará el entorno

<!-- @tip "Al importar un proyecto existente, Vibes lee el package.json para detectar el framework, las dependencias y los scripts disponibles." -->

## Desde el chat

También puedes describir lo que quieres crear directamente en cualquier chat activo. El agente generará todo el código necesario y lo organizará en la estructura del proyecto.

## Configuración previa

Antes de crear una app, puedes seleccionar:

- **Plantilla**: el framework base (React, Vue, Next.js, Astro...)
- **Sistema de diseño**: un archivo DESIGN.md con directrices visuales

Ambos se aplican automáticamente al crear el proyecto.
