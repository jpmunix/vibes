# Crear una Aplicación

Vibes ofrece múltiples formas de iniciar un nuevo proyecto.

## Desde una plantilla

1. Haz click en **Nueva Aplicación** en la barra lateral
2. Selecciona una plantilla del catálogo (React, Vue, Svelte, Next.js, Astro...)
3. Dale un nombre a tu proyecto
4. Vibes creará la estructura del proyecto y arrancará el servidor de desarrollo

## Desde una carpeta existente

Si ya tienes un proyecto en tu máquina:

1. Haz click en **Importar Aplicación**
2. Selecciona la carpeta de tu proyecto
3. Vibes detectará automáticamente el framework y configurará el entorno

## Desde el chat

También puedes describir lo que quieres crear directamente en el chat:

> "Crea una aplicación React con un formulario de contacto y una galería de imágenes"

La IA generará todo el código necesario y lo organizará en la estructura del proyecto.
