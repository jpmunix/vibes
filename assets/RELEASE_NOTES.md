## 🔥 Novedades principales

### Performance
- Correción del bug que bloqueaba el Agente inteligente cuando no habíamos establecido reglas de contexto

### Chat
- Añadida capacidad de colapsar el bloque de respuesta de la IA para buscar nuestros propios mensajes (funciona raruno)

- Capacidad de hacer undo y recuperar nuestro prompt y assets

### Notas
- Las notas ahora se pueden exportar a docx para poder abrirlas en las principales plataformas como word, libre office y google docs

### UI
- Mejoradas las herramientas de captura de pantalla
- Simplificada la barra de acciones de un chat

### Core
- Refactor del sistema de conocimiento automático para evitar ruido y falsos positivos del contexto aprendido de manera autónoma

### Herramientas
- Añadidas herramientas para poder controlar mejor nuestro repositorio Git
- Integración con supabase para poder examinar nuestra base de datos de manera sencilla
