# v8.7
*21 de mayo de 2026*

## Diseña tus propios Agentes de IA

Hasta ahora, Vibes te ofrecía un conjunto predefinido de agentes para interactuar con tu proyecto (como Build, Plan o Explore). A partir de esta versión, tienes **control total sobre las directivas de la IA** mediante el nuevo **Sistema de Agentes Personalizados**. 

Desde la pestaña de Ajustes, podrás diseñar tus propios agentes adaptados a tus flujos de trabajo específicos (por ejemplo, un agente experto en TypeScript y testing, un refactorizador ágil de estilos CSS, o un generador de documentación técnica).

<!-- @info "Cada agente personalizado que definas se asocia a un agente base nativo de OpenCode para heredar sus capacidades de ejecución de comandos, lectura de archivos o planificación." -->

### ¿Cómo configurarlos?
Al crear o editar un agente personalizado, dispones de las siguientes opciones:
1. **Nombre**: El identificador visual que verás en toda la aplicación.
2. **Comando Slash**: Un comando corto (sin la barra `/`) que sirve para invocarlo directamente desde la caja de chat.
3. **Agente Base**: Define si tu agente usará la base de `Build` (capacidades de edición y ejecución completas), `Plan` (modo restringido para proponer cambios) o `Explore` (modo lectura y análisis seguro).
4. **Modo de Prompt**:
   - **Aditivo**: Tus instrucciones personalizadas se añaden a continuación del system prompt nativo de Vibes.
   - **Reemplazar**: Tus instrucciones pisan por completo el prompt del sistema. Esto te permite tener un agente totalmente puro y con el comportamiento exacto que tú decidas.
5. **System Prompt**: Las directivas e instrucciones que guiarán la lógica del modelo de IA.

## Invocación rápida en Chat e Interfaz Reactiva

Hemos rediseñado el selector y la caja de texto del chat para integrar de forma nativa tus nuevos agentes personalizados de la manera más fluida posible:

### Selector de Modo con Diseño Esmeralda
El selector de modo del chat ahora renderiza de forma reactiva tanto los agentes nativos como los personalizados. Para diferenciarlos de un vistazo, los agentes diseñados por ti se muestran en un elegante color **verde esmeralda distintivo**. Al seleccionar cualquiera de ellos, la preferencia se persiste automáticamente para ese chat específico en la base de datos local.

### Comandos Slash (`/`) en Tiempo Real
Ya no necesitas abrir el selector cada vez que quieras cambiar de agente. La caja de chat (Lexical) ahora implementa un autocompletado inteligente de comandos slash. Al escribir una barra inclinada (`/`), Vibes te sugerirá instantáneamente todos los agentes disponibles. 

Puedes escribir `/mi-agente` y pulsar `Enter` o `Espacio` para que el input reconozca el comando, cambie el modo visualmente al agente correspondiente y prepare tu consulta para ejecutarse bajo esa configuración.

## Mensajes enriquecidos e interactivos para Commits en el Chat

Hemos rediseñado la forma en la que el chat muestra la confirmación cuando realizas un commit desde el panel rápido integrado. En lugar de un simple mensaje de texto plano, ahora se inyecta un componente visual nativo (widget) en el historial del chat. Esta tarjeta estructurada muestra el tipo de acción realizada (Commit o Commit & Push), el mensaje dividido visualmente entre título y cuerpo, y una lista interactiva de los archivos modificados. Ahora, al hacer clic en cualquiera de estos archivos desde el propio chat, se abrirá automáticamente en tu editor.
